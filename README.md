# 💬 Aplicación de Chat Cliente-Servidor (Java)

### 🎓 Contexto Académico
Este proyecto es desarrollado como parte de la formación profesional en la **Ingeniería en Tecnologías de la Información**, específicamente para la asignatura de **Programación Orientada a Objetos (POO)**. 

El objetivo es aplicar conceptos avanzados de desarrollo de software para resolver problemas de comunicación en tiempo real.

---

## 🚀 Tecnologías y Conceptos Implementados

En este desarrollo se integran los siguientes pilares de la computación:

* **Programación de Red (Sockets):** Establecimiento de comunicación síncrona/asíncrona bajo el protocolo TCP/IP.
* **Concurrencia (Multithreading):** Uso de hilos (`Threads`) para permitir que el servidor gestione múltiples clientes de forma simultánea sin bloquear la ejecución.
* **Interfaces Gráficas (GUI):** Diseño de ventanas interactivas utilizando la librería **Swing** de Java.
* **Persistencia de Datos:** Implementación de **SQLite** para el almacenamiento local de historiales de mensajes y gestión de usuarios.
* **Programación Orientada a Objetos:** Aplicación estricta de principios como encapsulamiento, herencia y polimorfismo.
